# Is Sorted

Checks whether an array is sorted in ascending order.

---

## ðŸ“Œ Description

The function `isSorted` checks if the given array of numbers
is sorted in non-decreasing (ascending) order.

If every element is less than or equal to the next one,
the function returns `true`. Otherwise, it returns `false`.

An empty array is considered sorted.

---

## ðŸ§ª Example

```js
isSorted([]);                       // true
isSorted([-22, -5, 0, 3, 9]);       // true
isSorted([3, 9, -3, 10]);           // false
isSorted([1, 3, 4, 1, 1]);           // false

```
